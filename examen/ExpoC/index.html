Langage C

C'est un des langages les plus utilisés.

L'histoire du langage C

Le langage C a été inventé au cours de l'année 1972 dans les laboratoire Bell dans l'État américain du New Jersey.
Il était développé en même temps que UNIX par Ken Thompson et Dennis Ritchie.

Ken Thompson avait développé un prédécesseur de C, le langage B.

Par ailleurs, le C est un langage très populaire. Il est utilisé pour programmer une grande partie des logiciels que vous connaissez.

Enfin, le langage C est un des langages les plus connus et les plus utilisés qui existent.
Il est très fréquent qu'il soit enseigné lors d'études supérieures en informatique.

Ce qu'il faut retenir :

Au tout début, on a commencé à inventer un langage de programmation appelé l'Algol.

Ensuite un nouveau langage est créé appelé le CPL,

qui évolua lui-même en BCPL,

qui prit ensuite le nom de langage B.

Puis un beau jour, le langage C fait son apparition.

Ce langage, s'il a subi quelques modifications, reste encore un des plus utilisés aujourd'hui.

Un peu plus tard, on a proposé des ameliorations pour l'appeler « C++ ».

Le langage C++ n'est en fait rien d'autre que le langage C avec des ajouts permettant de programmer d'une façon différente.
